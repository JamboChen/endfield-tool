# Contributing to Endfield Tool

Thank you for your interest in contributing! This is a small community project, and we welcome all contributions.

## Quick Start

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/endfield-calc.git`
3. **Create a branch**: `git checkout -b feature/your-feature-name`
4. **Make your changes** and test locally
5. **Commit**: `git commit -m "Add: brief description of your changes"`
6. **Push**: `git push origin feature/your-feature-name`
7. **Open a Pull Request** on GitHub

## Development Guidelines

### Code Style
- Use TypeScript for type safety
- Follow existing code patterns and formatting
- Add comments for complex logic (in English)
- Keep components small and focused

### Commit Messages
Use clear, descriptive messages:
- `Add: new feature description`
- `Fix: bug description`
- `Update: what was updated`
- `Refactor: what was refactored`

## Reporting Issues

Found a bug or have a suggestion? [Open an issue](https://github.com/JamboChen/endfield-calc/issues) with:
- Clear description of the problem/suggestion
- Steps to reproduce (for bugs)
- Screenshots if applicable
- Your browser/environment info (if relevant)

## Questions?

- Join our [Discord](https://discord.gg/6V7CupPwb6) for discussions
- Check existing issues before opening a new one
- Feel free to ask questions in your PR

## Code of Conduct

- Be respectful and constructive
- Focus on the project, not personal matters
- Help create a welcoming environment for everyone
